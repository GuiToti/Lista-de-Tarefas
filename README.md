Lista de Tarefas Simples

Este é um projeto básico de **Lista de Tarefas** desenvolvido com **HTML**, **CSS** e **JavaScript**. 

---

**Funcionalidades**

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista

---

**Tecnologias**

- HTML5
- CSS3
- JavaScript (Puro)
